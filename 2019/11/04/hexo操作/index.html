hexo文档

新建一篇文章

$ hexo new [layout]

layout = 布局，默认值为post 代表路径为source/_posts

生成文件&部署

$ hexo generate –deploy

$ hexo deploy –generate

简写

$ hexo g -d

$ hexo d -g
